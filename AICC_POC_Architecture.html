3 · Fusion Layer — From Raw Signals to Unified Event-State
The Fusion Layer is the POC's core differentiator. It converts a heterogeneous firehose into a single, time-aligned, context-enriched event-state vector that the AI Decision Engine can act on.
Responsibilities

What the Fusion Layer does

  • Normalize units, timebases, and coordinate systems across every feed.
  • Resolve entities — zones, cohorts, assets — into stable identifiers.
  • Grid space via geohash tiles for density, flow, and velocity math.
  • Align signals temporally with sliding windows and lag correction.
  • Vectorize into anonymous feature vectors — no PII ever persisted.
  • Enrich with weather, demographics, schedule, and historical priors.
  • Filter noise, dedup, and jitter before model inference.
Output contract

Unified Event-State Vector

For each zone and each tick, the Fusion Layer emits a structured vector the AI Engine consumes:

{ "event_id": "stampede_2026_d3", "zone": "GRANDSTAND_NE_B7", "ts": "2026-07-05T20:42:15Z", "density_ppsm": 4.8, // people / m² "flow_vec": [-0.62, 0.31], // m/s, heading "heat_index_f": 91, "demo_cardiac_mult": 1.7, "alcohol_velocity_idx": 0.82, "queue_len_m": 28, "nearest_ems_m": 140, "noise_db": 96, "sentiment_delta_5m": -0.23, "historical_prior": "elevated" }
